overflow 溢出处理

  1. visible 默认。内容不被裁剪，超出元素框的依然可以显示
  2. hidden 内容被裁剪，超出元素框的不显示
  3. scroll 内容被裁剪，但会添加滚动条以便查看其余内容
  4. auto 类似于scrool
