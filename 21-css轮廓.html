outline 轮廓简写

  1. outline: width style color
  2. outline: 宽度 样式 颜色

outline-style 轮廓样式

  1. dotted 点状轮廓
  2. dashed 虚线轮廓
  3. solid 实线轮廓
  4. double 双线轮廓
  5. none 无轮廓
  6. hidden 隐藏轮廓

outline-width 轮廓宽度

  1. thin 1px
  2. medium 3px
  3. thick 5px
  4. 一个特定的尺寸：以px，em为单位

outline-color 轮廓颜色
