border 边框属性速记

  1. border: border-width border-style border-color;
  2. border: 宽度 样式 颜色
  3. border: 5px solid red;
  4. border-style是必填的

四条边框属性分别速记

  1. border-top 上边
  2. border-left 左边
  3. border-right 右边
  4. border-bottom 下边

border-style 边框:

  1. dotted 点状边框
  2. dashed 虚线边框
  3. solid 实线边框
  4. double 双线边框
  5. none 无边框
  6. hidden 隐藏边框

可分别设置边框的四条边框:

  1. border-top-style 上边框
  2. border-right-style 右边框
  3. border-bottom-style 下边框
  4. border-left-style 左边框

分别设置四条边框的简写方法:

  1. border-style: 上边 右边 下边 左边
  2. border-style: 上边 左右边 下边
  3. border-style: 上下边 左右边

border-width 边框宽度:

  1. thin 薄
  2. medium 中
  3. thick 厚
  4. px或者em

border-width的简写方法与border-style的简写方法是一致的

  1. border-width: 5px 5px 5px 5px
  2. ...

border-color 边框颜色

border-color 的简写方法与border-style的简写方法也是一致的

  1. border-color: red blue yellow green
  2. ...
