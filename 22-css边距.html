margin

  1. auto：浏览器自动计算的边距（水平居中）
  2. length：以px，pt，cm等单位
  3. %：父元素宽度的百分比
  • margin: 上 右 下 左
  • margin: 上 左右 下
  • margin: 上下 左右
  • margin: 上下左右

css 边距

  1. margin-top 上边距
  2. margin-bottom 下边距
  3. margin-left 左边距
  4. margin-right 右边距

边距塌陷

  1. 两个相邻元素的边距会重合，只取最大的边距作为最终边距，并不会两个边距相加
  2. 边距塌陷只会发生在上下边距，不会发生在左右边距
